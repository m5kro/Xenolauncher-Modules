# Xenolauncher-Modules
The game engine module repository for [Xenolauncher](https://github.com/m5kro/Xenolauncher/).

Please read [module-docs.md](https://github.com/m5kro/Xenolauncher-Modules/blob/main/module-docs.md) for information on how to create a new module from scratch.

# Available Modules

| Name | Version | Author | License | Description |
| :----------------: | :--------: | :----------------: | :----------------: | :---- |
| EasyRPG | 1.1.0 | m5kro | GPL-3.0 | A module to launch RPG Maker 2000/2003 games with EasyRPG Player. |
| MacOS | 1.0.0 | m5kro | MIT | A default module to launch MacOS native games. |
| MKXP-Z | 1.1.0 | m5kro | GPL-3.0 | A module to launch MKXP-Z native games like RPG Maker XP, VX, and VX Ace. |
| NW.js | 1.3.2 | m5kro | GPL-3.0 | A module to launch NW.js native games like RPG Maker MV and MZ. |
| Wine | 0.1.0 | m5kro | GPL-3.0 | IN BETA! A module to launch Windows games using Wine. |
